# CRUD__React-JSON-Server-REST-API
```
git clone https://github.com/Hasul79/CRUD__React-JSON-Server-REST-API.git
cd CRUD__React-JSON-Server-REST-API
npm install
npm start 
```
### Json-server

```
npm i json-server --watch db.json --port 8000
```
